## 📋 About claude-bug-bounty

claude-bug-bounty helps you find security flaws in websites and apps. It works with AI to assist bug bounty hunters. The tool focuses on common issues like information leaks, broken access control, cross-site scripting (XSS), server-side request forgery (SSRF), OAuth security, GraphQL weaknesses, and large language model (LLM) injection vulnerabilities. It also helps generate detailed reports of your findings.

You don’t need to be a programmer to use this. The app runs on Windows and guides you through each step.
